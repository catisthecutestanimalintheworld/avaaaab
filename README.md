 Cygnus於1984年發售。正式型號為XC125，引擎使用124cc排氣量，以圓弧曲線為設計特徵的車體外觀。雖然並不是一台很人氣的車種，但卻是販賣相當久的商品，1988年的小改款後，持續在日本市場販售了10年以上。
